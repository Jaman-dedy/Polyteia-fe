## Getting Started

### Prerequisites

- Node.js: [Download and Install Node.js](https://nodejs.org/)
- npm or yarn

### Installation

1. **Clone the Repository:**

   ```bash
   git clone git@github.com:Jaman-dedy/Polyteia-fe.git


2. **Navigate to the Project Directory:**

```bash
cd your-project

```

3. **Install Dependencies**

```bash
yarn install

```

## Running the Server

1. **Locally**

```bash
yarn dev

```
